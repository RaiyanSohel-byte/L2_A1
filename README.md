# B7A1 - Advanced Problem Solving with TypeScript & OOP

This repository contains my solutions for the **B7A1** assignment. The goal of this assignment is to practice core TypeScript concepts, object-oriented programming, and problem-solving techniques by implementing different programming tasks and writing technical blog posts.

## Assignment Overview

The assignment focuses on:

- TypeScript functions
- Arrays and string manipulation
- Union types and type guards
- Generics
- Interfaces
- Classes and inheritance
- Object-Oriented Programming (OOP)
- Technical writing

All coding problems are completed in a single `solutions.ts` file, while the blog topics are written in separate Markdown files.

## Problems Solved

### Problem 1

Implemented `filterEvenNumbers()` to return only even numbers from an array.

### Problem 2

Implemented `reverseString()` to reverse a given string.

### Problem 3

Created a union type (`StringOrNumber`) and used type guards in `checkType()` to identify whether the input is a string or a number.

### Problem 4

Implemented a generic function `getProperty()` that safely returns an object's property value using TypeScript generics.

### Problem 5

Created a `Book` interface and implemented `toggleReadStatus()` to return a new object with an added `isRead` property.

### Problem 6

Implemented a `Person` class and a `Student` class using inheritance, along with a `getDetails()` method.

### Problem 7

Implemented `getIntersection()` to return the common elements between two arrays.
